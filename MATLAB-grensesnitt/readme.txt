For å starte GUI:
Åpne MATLAB, trykk på GUIDE på menylinjen.
Velg åpne eksisterende GUI og browse deg fram til LegoGui.fig.
I LegoGui trykkes det på Run, (Ctrl + T) eller trekantbutton på menylinje.


For å kjøre simuleringer velges simulator i tilkoblingsvinduet øverst til venstre i gui og trykk den grønne startknappen. Simuleringparametere kan velges i Gui og i nedtrekksmenyen "Edit", "Parameters". 

Tilkobling til roboten gjøres tilsvarende i tilkoblingsvinduet ved å velge rett serieport. Prøv først å pinge roboten, melding fra robot skal være: "robot responding". Lavnivåkommandoer(forward, stop, left, right, etc) kan nå utføres. Automatisk kartlegging og navigering velges ved å trykke på den grønne startknappen.
