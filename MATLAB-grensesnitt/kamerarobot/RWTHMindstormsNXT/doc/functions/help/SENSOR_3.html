SENSOR_3

Symbolic constant SENSOR_3 (returns 2)

Contents

  • Syntax
  • Description
  • Example
  • See also
  • Signature

Syntax

port3 = SENSOR_3()

Description

port3 = SENSOR_3() returns 2 as the value port3.

Example

  port3 = SENSOR_3()
  %result: >> port3 = 2
