MOTOR_B

Symbolic constant MOTOR_B (returns 1)

Contents

  • Syntax
  • Description
  • Example
  • See also
  • Signature

Syntax

portB = MOTOR_B()

Description

portB = MOTOR_B() returns 1 as the value portB.

Example

 portB = MOTOR_B()
 %result: >> portB = 1
