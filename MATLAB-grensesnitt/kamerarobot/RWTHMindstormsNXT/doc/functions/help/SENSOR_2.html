SENSOR_2

Symbolic constant SENSOR_2 (returns 1)

Contents

  • Syntax
  • Description
  • Example
  • See also
  • Signature

Syntax

port2 = SENSOR_2()

Description

port2 = SENSOR_2() returns 1 as the value port2.

Example

  port2 = SENSOR_2()
  %result: >> port2 = 1
