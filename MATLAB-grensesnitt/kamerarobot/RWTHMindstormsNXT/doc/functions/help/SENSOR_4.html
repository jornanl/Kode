SENSOR_4

Symbolic constant SENSOR_4 (returns 3)

Contents

  • Syntax
  • Description
  • Example
  • See also
  • Signature

Syntax

port4 = SENSOR_4()

Description

port4 = SENSOR_4() returns 3 as the value port4.

Example

  port4 = SENSOR_4()
  %result: >> port4 = 3
